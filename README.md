# Lime

> Application de messagerie professionnelle

## Descriptif

L’application est une plateforme de messagerie collaborative destinée aux équipes de travail. Elle permet aux utilisateurs de communiquer facilement, d’organiser les discussions par thématiques et de partager des informations en temps réel.

## Objectif

L'objjectif de l'application est d'offrir un outils qui permet de facilité la communication et la collaboration au sein des équipes, l'application permettrait d'eviter les echanges d'email, et centraliser toutes les discussions professionelles dans
un meme endroit

## Fonctionnalités

- Messagerie instantanée : les utilisateurs peuvent envoyer des messages en temps réel.

- Canaux de discussion : les conversations peuvent être organisées dans des canaux par projet, équipe ou sujet.

- Messages privés : possibilité d’échanger directement entre deux utilisateurs.

- Partage de fichiers : les utilisateurs peuvent envoyer des documents, images ou liens.

- Notifications : l’utilisateur reçoit des alertes lorsqu’il reçoit un message ou lorsqu’il est mentionné.

- Recherche : possibilité de rechercher des messages ou fichiers dans l’historique.

- Réactions et emojis : interaction rapide avec les messages
